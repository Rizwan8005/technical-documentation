Project Setup

  1. Download Visual Studio Code from: https://code.visualstudio.com/Download
  2. Download and install Node.js in your system from: https://nodejs.org/en/download
  3. Open project folder into VS Code.
  4. Press 'ctrl + j' to open the terminal in VS Code.
  5. Type ‘node -v’ then npm -v if it shows you the versions it means you have successfully installed the packages.
  6. Run command 'npm i' in the terminal to install all the packages required to run the app.
  7. Type npm start or yarn start for run project locally

React Application Deployment

  1. Press 'ctrl + j' to open the terminal in VS Code and run command 'npm run build', in the terminal, it will create a build folder.
  2. Convert the build folder into a zip folder.
  3. Sign in to your hosting, in my case my hosting is stackcp: https://www.stackcp.com
  4. After signing up click on File Manager in Web Files then click on public_html folder here remove unnecessary files and folders except admin folder, cointracker folder, .htaccess file and index.php file after deleting all files and folder click on below Uploads button.
  5. You will be redirected to this page: https://www.stackcp.com/services/6ff55fbb59b4085a/file-manager
  6. Now open your project folder, drag your build folder into the stackcp drag & drop circle.
  7. After uploading the build folder you will get build folder in the form of zip, right click on that folder and click on “Extract Here” after extracting build folder you will get build folder click on that folder copy all files and folder and then click on public_html file and paste here that files and folder.
  8. That's it! You have successfully deployed your ReactJS application.

Folder Structure for ReactJS (Frontend)

  • src: This is the main folder that contains all the source code files for your React.js project.
    • index.js: The entry point of your application where the ReactDOM renders your React components.
    • App.js: The root component of your application that serves as the container for all other components.
  • components: This folder contains reusable components that are used throughout your application.
  • pages: This folder contains components that represent different pages or views of your application.
    • Home.js: An example component representing the home page.
    • About.js: An example component representing the about page.
  • assets: This folder contains static assets such as images.
  • utils: This folder contains utility functions or helper classes used throughout your application like dropdown menu and modal etc.
  • Tailwind.config.js: This file contains all the tailwind customizations.
  • routes: This folder contains the routing configuration for your application.
  • store: This folder contains files related to state management (Redux).
    • slices: This folder contains all the slices each slice file holds its reducers and respective action creators and index.jsx file in store folder is used to combine all the slices in one file.
    • index.jsx: Combines all slices in one file.
  • Data: Holds dummy data for the application.
  • index.html: HTML template file for rendering the React application.
